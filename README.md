# Match Game

Match Game is a Phaser game engine to make line matching games. It allows a variety of game line matching games, from 3+ in a row, to different tiles.

## Quick Start

```
npm install
```

### Development

```
npm run dev
```

### Formatting

```
npm run format
```

### Build

```
npm run build
```
